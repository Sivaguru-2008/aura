# AURA Documentation Map

Welcome to the documentation for AURA (Adaptive Uncertainty-aware Reasoning Assistant) — a clinical intelligence platform designed for calibrated uncertainty, missing-evidence recommendation, and offline clinical decision support.

This directory serves as the unified, modern documentation hierarchy for the project.

## Directory Structure

* **[README.md](file:///e:/AURA/aura-main/docs/README.md)**: This file. A guide to the documentation map.
* **[ARCHITECTURE.md](file:///e:/AURA/aura-main/docs/ARCHITECTURE.md)**: Full architecture breakdown covering the 9-stage pipeline, event bus orchestrator, and classical-quantum stack.
* **[CURRENT_STATUS.md](file:///e:/AURA/aura-main/docs/CURRENT_STATUS.md)**: Real-time capability status detailing what is built, what is partial, and what is planned.
* **[PRODUCT_ROADMAP.md](file:///e:/AURA/aura-main/docs/PRODUCT_ROADMAP.md)**: The strategic vision, clinical applicability, and long-term roadmap.
